MoodDiary : Personal Journal & Mood Tracker Built with .NET 9, MAUI Blazor, SQLite

A privacy-first desktop journaling app featuring:

📓 Rich text journal entries with mood & tag tracking
📊 Analytics dashboard with mood trends & streaks
🔒 Entry locking with PIN protection
🌙 Light / Dark mode toggle
📅 Calendar view of writing history
📄 PDF/JSON export
